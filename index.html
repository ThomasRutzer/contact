Futurium

permanent exhibition

first release2019

key facts
  • react
  • mqtt

In fall of 2019 futurium, a museum which explores competing future scenarios, opened its doors in Berlin. art+com is lead agency for the exhibition design with its mix of analogue and digital media.

I was part of the team which has been responsible for a solid infrastructure as well as for multiple exhibits itself. The infrastructure includes automated deployment via ansible and a backend with multiple services around a mqtt-broker. That guarantees realtime communication across all the exhibits. Various sensors enhance the user interaction and are also connected to exhibits by mqtt.

Mntns

github data vizualisation

first release2018

key facts

Mntns is a self initiated web experience to visualize open github data for a username.

It fetches open github data of a useres and transforms them into a webGL based Mntns landscape. Several leves can be explored, each based on another data set.

Qurator

nlp service

first release2019

key facts
  • react
  • nlp

Qurator is a research collaboration focusing on content curation with AI technologies. The team at art+com built a prototypical nlp based service. I led the frontend development.

The service helps users to analyze large amounts of texts (e.g. their own text documents or Wikipedia articles) with a suitable interface. The backend component outputs a result which includes entities like geographical locations, historical events or personalities. Through the frontend component users can select entities and examine its occurrences. A variety of filters make the input text more accessible.
